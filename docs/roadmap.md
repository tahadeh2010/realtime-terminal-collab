# Roadmap

## Phase 1 - Foundation

Goal:
Create working backend skeleton.

Tasks:

* Project setup
* Folder structure
* Session model
* Client model
* Session manager

Deliverable:
Compile-able backend.

---

## Phase 2 - Realtime Layer

Goal:
WebSocket communication.

Tasks:

* WebSocket endpoint
* Connection handling
* Join session workflow
* Broadcast system

Deliverable:
Multiple clients connected.

---

## Phase 3 - Terminal Integration

Goal:
Real terminal execution.

Tasks:

* PTY integration
* Shell spawning
* Input forwarding
* Output streaming

Deliverable:
Shared terminal works.

---

## Phase 4 - Frontend

Goal:
Basic UI.

Tasks:

* Terminal component
* Session page
* Join flow

Deliverable:
Usable MVP.

---

## Phase 5 - Hardening

Goal:
Production readiness.

Tasks:

* Logging
* Metrics
* Error handling
* Tests

Deliverable:
Stable release candidate.
