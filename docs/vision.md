# Realtime Terminal Collaboration

## Vision

A lightweight platform that allows multiple users to collaborate around a shared terminal session in real time.

The goal of this project is not only to build a working application, but also to establish a development workflow that can be effectively managed using AI agents.

## Problem

Remote collaboration on terminal-based workflows is often difficult.

Existing solutions may be:

* Too heavy
* Self-hosting unfriendly
* Complex to extend
* Difficult to learn from

## Solution

Provide a web-based collaborative terminal where multiple users can join the same session and observe terminal output in real time.

## Target Users

* Developers
* DevOps engineers
* Students
* Mentors
* Pair-programming teams

## MVP Goals

* Create session
* Join session
* Shared terminal
* Real-time output synchronization
* Host-controlled input

## Success Criteria

* Multiple users can join one session
* Output synchronization latency remains low
* Session management is stable
* Architecture remains understandable by AI agents

## Non Goals (MVP)

* Authentication
* Billing
* Docker isolation
* Recording
* Multi-cursor editing
* Kubernetes deployment
